## 檢定系統

### 基本公式

```
行動成功率 = 對應屬性 × 20% + 難度修正
投擲 1D100，小於等於成功率即成功
```

### 難度等級

| 難度 | 修正值 | 適用情境 |
|------|--------|----------|
| 簡單 | +20% | 日常行動、基礎任務 |
| 普通 | +0% | 標準挑戰 |
| 困難 | -20% | 危險行動、技術任務 |
| 極難 | -40% | 生死攸關、超凡挑戰 |

### 特殊結果

| 骰值 | 結果 | 效果 |
|------|------|------|
| 01-05 | 大成功 | 完美達成，額外獎勵 |
| 96-00 | 大失敗 | 嚴重失誤，負面後果 |

## 戰鬥系統

### 回合流程

1. **先攻判定**：全員敏捷檢定，高者先行動
2. **行動階段**：攻擊/使用物品/移動/施法
3. **結算階段**：計算傷害、狀態效果

### 攻擊檢定

| 攻擊類型 | 使用屬性 |
|----------|----------|
| 近戰攻擊 | 力量 |
| 遠程攻擊 | 敏捷 |
| 魔法攻擊 | 智力 |

### 傷害計算

```
最終傷害 = 武器基礎傷害 + 屬性加成 - 敵人護甲
```

### 敵人行為模式

- 使用簡單描述配合骰子決定行動
- 範例：「哥布林投擲飛刀（敏捷檢定 -10%）」

## 資源管理

### 稜光瓶能量

- 每章 **3 次**調配機會
- 在**光之節點**可完全充能
- 用於調配各種藥水

### 負重系統

| 狀態 | 條件 | 效果 |
|------|------|------|
| 正常 | 總重量 ≤ 負重上限 | 無懲罰 |
| 超重 | 總重量 > 負重上限 | 所有檢定 -20% |
