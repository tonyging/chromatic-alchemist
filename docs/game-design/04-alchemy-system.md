# 藥水煉金系統

---

## 調配機制

### 調配時機
- 僅能在**營地休息**時進行
- 消耗稜光瓶能量（每章 3 次機會）

### 調配檢定

```
成功率 = 感知 × 20% + 配方難度修正
```

### 失敗後果
- 材料損失 **50%**
- 不消耗稜光瓶能量
