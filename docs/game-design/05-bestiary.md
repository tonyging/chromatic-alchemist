## 資料格式說明

```
【敵人名稱】
HP: 生命值 | 迴避: 迴避率% | 護甲: 傷害減免
攻擊: 攻擊名稱（傷害，特殊效果）
特性: 特殊能力
弱點: 克制方式
```

## 敵人行為 AI 指南

### 簡化原則
1. 每種敵人最多 2-3 種行動
2. 使用 1D6 決定行動：
   - 1-3：基礎攻擊
   - 4-5：特殊攻擊
   - 6：使用特性

### 戰鬥節奏控制
- 小型戰鬥：3 回合內結束
- Boss 戰：5-8 回合
- 連續失敗可降低敵人傷害
