# 遊戲流程與 GM 指南

---

## 章節流程結構

每章按以下順序進行：

```
1. 章節開場描述
   ↓
2. 探索階段（收集材料、遭遇事件）
   ↓
3. 核心挑戰（戰鬥或謎題）
   ↓
4. Boss 戰
   ↓
5. 獲得聖杯碎片
   ↓
6. 章節結算（經驗、新配方、屬性提升）
```

## GM 運作建議

### 1. 準備檢定表

**預先計算常見成功率：**

| 屬性值 | 簡單 (+20%) | 普通 (+0%) | 困難 (-20%) | 極難 (-40%) |
|--------|-------------|------------|-------------|-------------|
| 1 | 40% | 20% | 0% | 0% |
| 2 | 60% | 40% | 20% | 0% |
| 3 | 80% | 60% | 40% | 20% |
| 4 | 95% | 80% | 60% | 40% |
| 5 | 95% | 95% | 80% | 60% |

### 2. 簡化戰鬥

- **小型戰鬥**：3 回合內結束
- **中型戰鬥**：4-5 回合
- **Boss 戰**：5-8 回合
- 避免拖沓，保持節奏

### 3. 給予提示

```
智力檢定成功時可提供：
- Boss 弱點線索
- 隱藏通道位置
- 陷阱警告
- 敵人行動模式
```

### 4. 靈活調整

**玩家連續失敗時：**
- 降低下一次檢定難度
- 提供額外提示
- 減少敵人傷害輸出
- 給予免費回復機會

**玩家過於順利時：**
- 增加隨機事件
- 引入額外挑戰
- 但不要無理由懲罰

### 5. 記錄道具

**每章末整理：**
```
【背包清單】
- 紅稜石：__
- 綠稜石：__
- 藍稜石：__
- 紅光藥水：__
- 綠光藥水：__
- 藍光藥水：__
- 特殊材料：__________
- 當前負重：__/__
```

## 核心原則

> **保持敘事流暢，骰子只用於關鍵時刻**

### 何時擲骰

**需要擲骰：**
- 危險行動（跳過深淵、躲避攻擊）
- 戰鬥檢定
- 藥水調配
- 重要的社交互動

**不需要擲骰：**
- 日常移動
- 簡單對話
- 明顯能成功的行動
- 純敘事推進

## 常見情境處理

### 玩家想做意料外的事

```
1. 判斷是否合理
2. 若合理，指定對應屬性與難度
3. 讓玩家擲骰
4. 根據結果即興發揮
```

### 玩家卡關時

```
1. 提供感知/智力檢定機會
2. 成功則給予提示
3. 失敗仍可給予模糊提示
4. 絕對避免死鎖
```

### 戰鬥陷入僵局

```
1. 引入環境變化（地形崩塌、援軍等）
2. 敵人展露弱點
3. 提供逃跑/談判機會
4. 必要時直接敘事推進
```
