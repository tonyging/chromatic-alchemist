# 異常狀態定義

---

## 狀態分類

| 分類 | 說明 | 包含狀態 |
|------|------|----------|
| 持續傷害 | 每回合造成傷害 | 中毒、灼燒 |
| 行動限制 | 影響行動能力 | 定身、冰凍、束縛、混亂 |
| 能力削弱 | 降低屬性或檢定 | 虛弱、致盲、護甲破損 |
| 特殊狀態 | 其他效果 | 隱身、護盾、再生 |

---

## 負面狀態

### 中毒

| 項目 | 內容 |
|------|------|
| 效果 | 每回合結束時受到 3 點傷害 |
| 持續 | 3 回合，或直到解除 |
| 來源 | 虛無教團刺客、毒孢子雲、毒沼地形 |
| 解除 | 青光淨化藥、解毒草、光之節點休息 |
| 疊加 | 不可疊加，重複施加時重置持續時間 |

### 灼燒

| 項目 | 內容 |
|------|------|
| 效果 | 每回合結束時受到 5 點傷害 |
| 持續 | 2 回合 |
| 來源 | 火焰吐息、岩漿地形 |
| 解除 | 青光淨化藥、跳入水中 |
| 疊加 | 不可疊加 |

### 定身

| 項目 | 內容 |
|------|------|
| 效果 | 無法移動，但可攻擊和使用物品 |
| 持續 | 2 回合，或直到解除 |
| 來源 | 藤蔓纏繞、陷阱 |
| 解除 | 力量檢定（困難）、青光淨化藥、隊友協助 |
| 疊加 | 不可疊加 |

### 冰凍

| 項目 | 內容 |
|------|------|
| 效果 | 完全無法行動，受到傷害時傷害 +50% |
| 持續 | 1 回合，或受到火焰傷害解除 |
| 來源 | 寒冰風暴、冰霜巨人 |
| 解除 | 受到任意傷害（含火焰）、保溫藥水預防、抗火藥水預防 |
| 疊加 | 不可疊加 |
| 特殊 | 被冰凍時受到的傷害會同時解除冰凍 |

### 束縛

| 項目 | 內容 |
|------|------|
| 效果 | 所有檢定 -20%，無法逃跑 |
| 持續 | 直到解除 |
| 來源 | 暗影鏈鎖、陷阱 |
| 解除 | 敏捷檢定（困難）、青光淨化藥 |
| 疊加 | 不可疊加 |

### 混亂

| 項目 | 內容 |
|------|------|
| 效果 | 行動時投 1D6：1-2 攻擊自己，3-4 原地發呆，5-6 正常行動 |
| 持續 | 2 回合 |
| 來源 | 相位打擊、幻象攻擊 |
| 解除 | 青光淨化藥、受到傷害有 50% 機率解除 |
| 疊加 | 不可疊加 |

### 虛弱

| 項目 | 內容 |
|------|------|
| 效果 | 力量、敏捷檢定 -20% |
| 持續 | 3 回合 |
| 來源 | 黑暗吞噬、長期飢餓 |
| 解除 | 紅光藥水、休息 |
| 疊加 | 不可疊加 |

### 致盲

| 項目 | 內容 |
|------|------|
| 效果 | 攻擊檢定 -30%，無法發現隱藏物品 |
| 持續 | 2 回合 |
| 來源 | 閃光攻擊、迷霧環境 |
| 解除 | 驅散迷霧藥水、時間經過 |
| 疊加 | 不可疊加 |

### 護甲破損

| 項目 | 內容 |
|------|------|
| 效果 | 護甲減傷效果無效 |
| 持續 | 當前戰鬥結束，或直到休息修復 |
| 來源 | 腐蝕吐息、特殊攻擊 |
| 解除 | 戰鬥結束後休息 |
| 疊加 | 不可疊加 |

## 正面狀態

### 再生

| 項目 | 內容 |
|------|------|
| 效果 | 每回合開始時回復 5 HP |
| 持續 | 3 回合 |
| 來源 | 綠光藥水 |
| 疊加 | 可疊加持續時間，但回復量不疊加 |

### 護盾

| 項目 | 內容 |
|------|------|
| 效果 | 吸收接下來 15 點傷害 |
| 持續 | 直到吸收完畢或戰鬥結束 |
| 來源 | 特殊藥水、環境效果 |
| 疊加 | 不可疊加，新護盾覆蓋舊護盾 |

### 隱身

| 項目 | 內容 |
|------|------|
| 效果 | 敵人無法主動攻擊，首次攻擊必定命中且傷害 +50% |
| 持續 | 直到攻擊或被偵測 |
| 來源 | 洋紅相位藥、環境掩護 |
| 疊加 | 不可疊加 |

### 抗性（火/冰/毒）

| 項目 | 內容 |
|------|------|
| 效果 | 對應屬性傷害減半，免疫對應異常狀態 |
| 持續 | 1 章節或當前戰鬥 |
| 來源 | 抗火藥水、保溫藥水、特殊裝備 |
| 疊加 | 不可疊加 |

## 狀態互動規則

### 互斥狀態

| 狀態 A | 狀態 B | 結果 |
|--------|--------|------|
| 冰凍 | 灼燒 | 兩者互相抵消 |
| 隱身 | 任意傷害/攻擊 | 隱身解除 |
| 再生 | 中毒 | 兩者同時生效，淨回復 = 5 - 3 = 2 |

### 優先順序（回合結束時）

```
1. 再生效果（回復 HP）
2. 持續傷害（中毒、灼燒）
3. 狀態持續時間 -1
4. 檢查狀態是否結束
```

## GM 注意事項

1. **簡化處理**：一般戰鬥中，同時存在超過 2 種負面狀態時，可詢問玩家是否簡化為「重傷狀態」（所有檢定 -20%，每回合 5 傷害）

2. **狀態提示**：玩家被施加狀態時，應明確告知效果和解除方式

3. **Boss 豁免**：Boss 對定身、冰凍等控制效果持續時間減半（最少 1 回合）
