# NPC 資料庫

---

## NPC 分類

| 類型 | 說明 | 互動方式 |
|------|------|----------|
| 主線 NPC | 推動劇情的關鍵角色 | 對話、任務 |
| 支線 NPC | 提供支線任務或資訊 | 對話、救助 |
| 商人 | 買賣物品 | 交易 |
| 隨機遭遇 | 探索時隨機出現 | 多種 |

## NPC 互動指南

### 對話檢定

| 情境 | 檢定 | 成功效果 |
|------|------|----------|
| 套取情報 | 智力（普通） | 獲得額外線索 |
| 討價還價 | 智力（普通） | 交易折扣 |
| 說服對方 | 智力（困難） | 改變 NPC 態度 |
| 察覺謊言 | 感知（困難） | 識破欺騙 |

### 好感度系統（簡化）

不使用數值追蹤，改用標記：
- **友好**：救助/幫助後
- **中立**：初始狀態
- **敵對**：傷害/背叛後

友好 NPC 可能在後續章節提供支援。
