# 謎題庫

---

## 謎題設計原則

1. **難度漸進**：序章簡單，後期複雜
2. **多解法**：力量破解 / 智力解謎 / 道具繞過
3. **不死鎖**：永遠有提示機制
4. **獎勵明確**：解謎後立即給予回饋

---

## 謎題類型

| 類型 | 說明 | 主要檢定 |
|------|------|----------|
| 符文謎題 | 按正確順序啟動符文 | 智力 |
| 機關謎題 | 操作機械裝置 | 智力/感知 |
| 邏輯謎題 | 推理正確答案 | 智力 |
| 光學謎題 | 操控光線路徑 | 感知 |
| 環境謎題 | 利用環境元素 | 感知/力量 |

## 序章謎題

### 燈塔光源謎題

**場景**：廢棄燈塔的頂層

**描述**：
```
燈塔中央有三面鏡子，分別刻著紅、綠、藍的符文。
牆上有一道被封印的門，門上發出微弱的白光。
地上的石板寫著：「三色合一，光明重現。」
```

**解法**：
1. 調整三面鏡子，讓光線交匯在門的中央
2. 智力檢定（簡單 +20%）確認正確角度

**提示機制**：
- 失敗後，守護靈會說：「紅色在左，藍色在右，綠色居中...」

**獎勵**：門開啟，獲得稜光瓶

**替代解法**：
- 力量檢定（困難 -20%）強行破門 → 成功但稜光瓶能量減半

## 第一章謎題

### 熔岩橋謎題

**場景**：火山內部的岩漿河

**描述**：
```
面前是一條岩漿河，河中有五塊浮動的岩石平台。
平台上分別刻著數字：3, 1, 4, 1, 5
每塊平台只能踩一次，踩錯會沉入岩漿。
岸邊的石碑寫著：「圓周之始，智者之路。」
```

**解法**：
- 按圓周率順序踩踏：3 → 1 → 4 → 1 → 5
- 智力檢定（普通 +0%）可識別提示

**失敗後果**：
- 踩錯 → 敏捷檢定（普通）跳回，失敗受 10 點火焰傷害

**獎勵**：
- 到達對岸，發現紅稜石 ×3

**替代解法**：
- 洋紅相位藥傳送 → 直接通過
- 繩索 + 敏捷檢定（困難）→ 盪過去

### 礦坑機關門

**場景**：礦工被困的區域入口

**描述**：
```
巨大的石門上有三個凹槽，分別是紅、黃、橙色。
旁邊散落著各色稜石。
門框上刻著：「熱之組合。」
```

**解法**：
- 紅 + 黃 = 橙（暖色系）
- 放入紅稜石、黃稜石（或紅+綠混合），門開啟
- 感知檢定（普通）可察覺顏色規律

**提示**：
- 鐵鬚隊長（如果對話過）會說：「聽說要放暖色的石頭...」

**獎勵**：
- 進入礦坑，觸發救助礦工支線

## 第二章謎題

### 森林迷宮

**場景**：翠綠森林深處

**描述**：
```
四條道路，入口分別有樹木標記：
- 北：三片葉子
- 東：兩片葉子
- 南：四片葉子
- 西：一片葉子

地上的青苔拼出文字：「生命始於一，綻放於三。」
```

**解法**：
- 西（1）→ 北（3）→ 走出迷宮
- 智力檢定（普通）理解提示

**錯誤路徑**：
- 任何其他組合 → 回到原點，遭遇隨機敵人

**獎勵**：
- 找到古樹神殿入口

### 毒沼淨化謎題

**場景**：毒素源頭

**描述**：
```
三個水池，分別呈現紅色（毒）、藍色（冷）、黃色（酸）。
中央有一個空的石盆。
石碑：「萬物歸於自然之色。」
```

**解法**：
- 將綠稜石放入石盆 → 三池毒素被中和
- 感知檢定（普通）可感應正確材料

**錯誤嘗試**：
- 放入其他稜石 → 毒霧噴發，體質檢定或中毒

**獎勵**：
- 淨化成功，樹靈恢復理智（和平路線前置）

## 第三章謎題

### 魔法圖書館密碼

**場景**：蒼穹浮空城圖書館

**描述**：
```
一本被鎖住的古書，鎖上有四個轉盤，每個可選 0-9。
旁邊有一張紙條：
「光裂之年，聖杯破碎。
 今日之前，五百寒暑。」
```

**解法**：
- 遊戲設定年份（假設當前 500 年）
- 光裂之災發生在 500 年前 = 第 0 年
- 密碼：0000
- 智力檢定（困難 -20%）可推算

**提示**：
- 艾克斯（圖書館管理員）：「記錄顯示，光裂之災發生在紀元元年。」

**獎勵**：
- 獲得索倫的日記，揭露其身份

### 魔像啟動謎題

**場景**：機械工坊

**描述**：
```
一個損壞的運輸魔像，胸口有三個能量槽。
槽位標記：○ □ △
地上有三塊能量水晶：紅（○）、藍（□）、綠（△）
但水晶旁的標籤被打亂了。
```

**解法**：
- 紅 → △、藍 → ○、綠 → □（形狀提示顏色首字母）
- 感知檢定（普通）注意到標籤墨跡

**錯誤後果**：
- 魔像暴走，強制戰鬥（弱化版魔像守衛，HP 50）

**獎勵**：
- 魔像啟動，可騎乘快速移動本章節

## 第五章謎題

### 冰封日記

**場景**：冰窟深處

**描述**：
```
一本日記被封在冰塊中。
冰塊上有一行字：「溫暖的記憶能融化一切。」
直接用火會損壞日記。
```

**解法**：
- 使用紅光藥水（生命之熱）慢慢融化
- 或：抗火藥水 + 小型火焰 = 控溫融化
- 感知檢定（困難 -20%）想到方法

**錯誤嘗試**：
- 直接用火 → 日記部分損毀，只能獲得片段資訊

**獎勵**：
- 完整的光裂之災日記，揭露索倫的悲劇

**場景**：洋紅裂隙

一條走廊，左右牆壁是鏡子。

**提示**：

**獎勵**：

## 第七章謎題

### 最終封印

**場景**：虛無聖殿祭壇

**描述**：
```
七個基座，需要放置七塊聖杯碎片。
但基座順序似乎很重要。
祭壇中央的刻文：「光譜之序，紅始紫終。」
```

**解法**：
- 按光譜順序放置：紅 → 橙 → 黃 → 綠 → 青 → 藍 → 紫
- 智力檢定（普通）回憶光譜知識

**錯誤後果**：
- 順序錯誤 → 暗影能量反噬，全員受 15 傷害

**獎勵**：
- 封印解除，可以挑戰索倫
- 自動獲得白光聖藥配方

## GM 謎題指南

### 提示給予原則

1. **第一次失敗**：給予環境提示
2. **第二次失敗**：NPC 或道具提示
3. **第三次失敗**：直接給出部分答案
4. **完全卡關**：允許力量/道具繞過

### 難度調整

| 玩家類型 | 調整方式 |
|----------|----------|
| 喜歡解謎 | 減少提示，增加獎勵 |
| 不擅解謎 | 增加提示，允許跳過 |
| 混合團隊 | 提供多種解法 |

### 謎題跳過規則

如果玩家實在無法解開：
- 消耗資源（金幣/材料）可繞過
- 或強制戰鬥代替（打敗守衛 = 通過）
