## 快速參考

### 核心公式
```
成功率 = 屬性 × 20% + 難度修正
HP = 20 + (力量 × 2)
MP = 10 + (智力 × 2)
```

### 難度修正
- 簡單: +20% | 普通: +0% | 困難: -20% | 極難: -40%

### 特殊骰值
- 01-05: 大成功 | 96-00: 大失敗
