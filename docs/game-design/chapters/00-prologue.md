## 場景一：夢境開場

### 開場描述

```
【朗讀給玩家】

黑暗。

你聽見師父的聲音，彷彿從很遠的地方傳來。

「記住，孩子...光與暗從不對立...」

聲音漸漸模糊。

「三稜聖杯...碎片...你必須...」

你猛然醒來。

冷汗浸濕了後背。窗外，殘月如鉤。
桌上，師父的最後一封信靜靜躺著。
信封上只有一行字：「廢棄燈塔，答案在那裡。」
```

### 玩家行動

**讀信**（自動觸發）：
```
親愛的學徒：

當你讀到這封信時，我已經不在了。
五百年前的光裂之災，聖杯碎成七片。
我花了一輩子尋找，只找到了線索。

去廢棄燈塔，那是折光師的起源之地。
稜光瓶在那裡等著你。

記住我教你的一切。
你是最後的折光師學徒。
世界的命運，交給你了。

願光明指引你的道路。
——艾爾文
```

### 過渡

```
你收起信，整理好行囊。
天亮之前，你必須出發。
```

## 場景三：燈塔外部

### 描述

```
【朗讀給玩家】

經過一天的跋涉，你終於來到了廢棄燈塔。

這座燈塔矗立在海岬盡頭，早已被世人遺忘。
石牆爬滿了藤蔓，窗戶破碎，海風從裂縫中呼嘯而過。
但奇怪的是，塔頂似乎還有微弱的光芒閃爍。

燈塔大門半開，黑暗在門後等待。

你可以：
A. 直接進入燈塔
B. 先觀察周圍環境（感知檢定）
C. 檢查門上是否有陷阱（感知檢定）

請選擇：____
```

### 選項結果

**A. 直接進入**：
- 進入場景四

**B. 觀察周圍**：
- 感知檢定（簡單 +20%）
- 成功：「你注意到燈塔旁有一個被藤蔓覆蓋的小棚子。裡面可能有什麼。」
  - 可探索小棚，獲得紅稜石 ×1
- 失敗：「周圍看起來很普通，只有岩石和海浪。」

**C. 檢查陷阱**：
- 感知檢定（普通 +0%）
- 成功：「門上沒有陷阱，但你注意到門框上有褪色的符文。這是折光師的標記。」
- 失敗：「門看起來只是一扇普通的老舊木門。」

## 場景七：燈塔頂層

### 描述

```
【朗讀給玩家】

你來到燈塔頂層。

這裡曾是折光師的工作室。儘管塵封多年，光學儀器仍閃爍著微弱的光芒。

房間中央有三面鏡子，分別刻著紅、綠、藍的符文。
對面牆上有一道被光芒封印的門，門上的符文緩緩流動。
門前站著一個淡金色的人形光影——燈塔的守護靈。

守護靈開口說話，聲音如風鈴般清脆：

「年輕的折光師...你終於來了。
 艾爾文說你會來的。
 要打開這扇門，你必須讓三色之光合一。」
```

### 謎題：光源對齊

```
【謎題：三色合一】

三面鏡子可以旋轉，調整光線方向。
門上有一個圓形標記，需要三道光線同時照射。

當前狀態：
- 紅鏡：指向左
- 綠鏡：指向前
- 藍鏡：指向右

你可以：
1. 調整紅鏡方向
2. 調整綠鏡方向
3. 調整藍鏡方向
4. 請求守護靈提示（智力檢定）
5. 嘗試其他方法

請選擇：____
```

### 謎題解法

**正確配置**：三面鏡子都指向門上的圓形標記

```
【智力檢定提示】（成功時）

你仔細觀察，發現：
- 每面鏡子發出的光都是單色的
- 門上的標記是白色的
- 紅 + 綠 + 藍 = 白

答案很簡單——讓所有光線匯聚到同一點。
```

**解謎成功**：
```
三道光線在門上的標記處交匯。
紅、綠、藍混合成耀眼的白光。

封印解除。門緩緩打開。

守護靈微笑：「很好。你有成為折光師的資質。」
```

**強行破門**（力量檢定，困難 -20%）：
```
[成功]
你用武器砸開了門。門碎裂的瞬間，一股能量反噬。
你受到 5 點傷害。
守護靈嘆息：「年輕人，有時候用腦比用力更重要。」

[失敗]
門紋絲不動。封印反彈了你的力量。
你受到 3 點傷害。
```

## 場景八：獲得稜光瓶

### 描述

```
【朗讀給玩家】

門後是一個小型密室。

牆上陳列著古老的折光師遺物——但大多已經碎裂或褪色。
唯獨中央的展示台上，有一個完好無損的水晶瓶。

那是稜光瓶。

瓶身透明，內部流動著七彩的光芒。這是折光師調配藥水的核心工具。

守護靈的聲音在身後響起：

「這是艾爾文留給你的。
 稜光瓶能儲存光之能量，用於調配各種藥水。
 每章你有三次調配機會，在光之節點可以充能。」

你獲得了【稜光瓶】。
```

### 教學：藥水調配

```
【教學：藥水調配】

守護靈繼續說道：

「讓我教你基礎的調配方法。」

【藥水調配規則】
1. 只能在營地休息時調配
2. 每章有 3 次調配機會
3. 調配需要進行感知檢定
4. 失敗會損失 50% 材料

【基礎配方】（現在學會）
- 紅光藥水：紅稜石 ×1 → 回復 10 HP
- 綠光藥水：綠稜石 ×1 → 回復 5 HP/回合 ×3
- 藍光藥水：藍稜石 ×1 → 回復 10 MP

「拿好這些材料，你會需要的。」

獲得：
- 紅稜石 ×3
- 綠稜石 ×3
- 藍稜石 ×3
```
